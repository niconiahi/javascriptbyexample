Javascript by example: objects

an object allows you to group pieces of data to represent something

objects are composed of key-value pairs

 
const object = {
  key: "value",
}
console.log("object", object)

“person” is an object which has two keys: “name” which value is “Mario” and “surname” which value is “Baracuz”

 
const person = {
  name: "Mario",
  surname: "Baracuz",
}
console.log("person", person)
 
$ deno run objects.js
object { key: "value" }
person { name: "Mario", surname: "Baracuz" }
